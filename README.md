Play the game:
https://mathieuviart.github.io/legrandquiz/

The strategic geography game — 8 countries, 8 categories, the lowest score possible.

Each turn, match the displayed country to a category.
Your score = the country's value in that category (rank, rate, count…).
Each category can only be used once.
You have 20 seconds per country. Otherwise: +200 penalty points!
Goal: minimize your total score!

Supported language:
FR - EN - UA
